monitor_facade.py is an Inksnek (https://github.com/funnypolynomial/inksnek) file to create a simple laser-cut monitor facade which clamps onto the Uno. 
monitor_facade.svg is the file to send to the laser cutter (assumes 3mm material). BLUE is CUT and RED is ETCH. The lines are very thin for sending the the laser cutter.

ICSP-RTC.zip is a Gerber for a small board which is soldered to the Uno ICSP pins and mates to a small RTC board, all sitting between the Uno and the LCD Shield. 
See https://hackaday.io/project/168896-i-c-s-p-r-t-c
